# Release v0.0.2

Release date: 2026-02-27

## Published Packages

- `flutterxel` 0.0.2: https://pub.dev/packages/flutterxel
- `flutterxel_tools` 0.0.2: https://pub.dev/packages/flutterxel_tools

Git tag:

- `v0.0.2`

## Highlights

- Added polished sample game projects under `examples/`:
  - `star_patrol`
  - `pixel_puzzle`
  - `void_runner`
  - `cosmic_survivor`
- Improved runtime robustness in the plugin runtime layer:
  - stronger native binding/fallback behavior
  - improved runtime state reset handling
  - expanded API-surface regression test coverage
- Improved ABI contract check script compatibility for environments without `rg`.
- Added a documentation site structure under `docs/`:
  - MkDocs + Material setup
  - API docs generated automatically from source via `dart doc`
  - manual GitHub Pages deployment workflow (`workflow_dispatch`)

## Upgrade Notes

### For package users

Update your dependency to:

```yaml
dependencies:
  flutterxel: ^0.0.2
```

### For repository examples

Example projects now target the published package version (`^0.0.2`) instead of a local path dependency.
