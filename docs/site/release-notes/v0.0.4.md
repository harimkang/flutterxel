# Release v0.0.4

Release date: 2026-03-04

## Published Packages

- `flutterxel` 0.0.4: https://pub.dev/packages/flutterxel
- `flutterxel_tools` 0.0.4: https://pub.dev/packages/flutterxel_tools

Git tag:

- `v0.0.4`

## Highlights

- Added TMX import support for square tile sizes that are integer multiples of `8` in `Tilemap.fromTmx(...)` / `Tilemap.from_tmx(...)`.
- Added normalization from larger TMX tiles (for example `16x16`) into flutterxel's internal `8x8` tile grid.
- Added validation for unsupported TMX map tile sizes (non-positive, non-square, or not divisible by `8`).
- Added regression tests for:
  - `16x16` normalization into `8x8` tile coordinates
  - invalid tile-size rejection via `FormatException`

## Upgrade Notes

### For package users

- Existing `8x8` TMX behavior remains unchanged.
- For larger TMX tiles, imported `Tilemap.width` / `Tilemap.height` scale by the tile-size ratio.
  - Example: a `1x1` map with `16x16` tiles imports as a `2x2` internal tilemap.

### For tooling users

- No functional CLI changes in `flutterxel_tools` for this release.
