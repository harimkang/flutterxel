# Release v0.0.13

Release date: 2026-03-06

## Published Packages

- `flutterxel` 0.0.13: https://pub.dev/packages/flutterxel
- `flutterxel_tools` 0.0.13: https://pub.dev/packages/flutterxel_tools

Git tag:

- `v0.0.13`

## Highlights

- Added bytes-based detached image loading:
  - `Image.fromBytes(...)`
  - `image.loadBytes(...)`
- Global `blt(...)` now accepts detached `Image` sources, including atlases created from bytes.
- Added destination-rect nearest-neighbor blits:
  - `bltEx(...)` / `blt_ex(...)`
  - `Image.bltEx(...)` / `Image.blt_ex(...)`
- Added widget-layer pointer sampling via:
  - `FlutterxelView(onPointerSample: ...)`
  - `FlutterxelPointerSample`

## Upgrade Notes

### For package users

- You can load encoded PNG bytes directly into detached images without a temporary file path.
- Detached atlases no longer need a resource-bank hop before rendering to the screen.
- `bltEx(...)` uses nearest-neighbor scaling and supports destination-space `pivotX` / `pivotY`.
- `FlutterxelView.onPointerSample` is observational; existing `btn(...)` / `btnv(...)` input mapping remains unchanged.
